# Sample Test Data

This directory contains sample data for testing the Legal Memory ingestion pipeline.

## Files

### WhatsApp Exports
- **sample_whatsapp.txt** — iOS format WhatsApp export with legal discussion examples
- **sample_whatsapp_android.txt** — Android format WhatsApp export
- **sample_whatsapp_ios.txt** — Additional iOS format export (generated by script)

WhatsApp exports can be generated from any WhatsApp chat by:
1. Open WhatsApp chat
2. More options → Export chat → Without media
3. Share the .txt file to this directory

### PDF Documents
- **sample_contract.pdf** — Sample service agreement
- **sample_nda.pdf** — Sample non-disclosure agreement

## Format Examples

### iOS Format
```
[MM/DD/YY, HH:MM:SS] Name: message text
[05/15/2024, 09:30:45] Alice: This is a message
```

### Android Format
```
MM/DD/YYYY, HH:MM - Name: message text
05/20/2024, 10:15 - Alice: This is a message
```

## Generating Test Data

Run the test data generation script:

```bash
python scripts/generate_test_data.py
```

This will create:
- Additional WhatsApp exports
- Sample PDF files (if reportlab is installed)

## Using Test Data

### Option 1: Upload via Web UI
1. Start the services:
   ```bash
   docker compose up -d
   ```

2. Run the Streamlit app:
   ```bash
   streamlit run main.py
   ```

3. In the sidebar, upload files from this directory

### Option 2: Ingest Programmatically
```python
from pathlib import Path
from app.ingest import ingest_whatsapp

result = ingest_whatsapp("sample_data/sample_whatsapp.txt")
print(f"Ingested {result['chunks']} chunks from {result['messages']} messages")
```
